### 利用した技術

- TypeScript　（AltJS）
- React / react-router-dom （JavaScript ライブラリ）
- Firebase Authentication / Cloud Firestore　（Baas：Backend as a Service）
- Chakra UI　（React UI コンポーネントライブラリ）

### 要件

- トレーニング内容を簡単に記録できる。
- トレーニングのメニューを登録できる。
- 複数ユーザーがログインして使える。

### 機能

- トレーニング履歴の登録、編集、削除。
- 履歴の年月またはメニューでの絞り込み。
- トレーニングメニューの登録、編集、削除。
- 会員登録、ログイン、ログアウト。
